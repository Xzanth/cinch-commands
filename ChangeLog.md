### 0.1.1 / 2016-08-15

* Xzanth's modifications:
  * Arguments now a list of hashes:
    * name specifies name of argument
    * format its format
    * optional whether the argument is optional or not
  * `!help` command can now respond through different means:
    * notify
    * query
    * reply
